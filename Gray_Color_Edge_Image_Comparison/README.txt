Contents and Execution instructions:

hw3.py
-run with: python3 hw3.py
-see cmd prompt for computation status
-the first window to show up will be the Quiver
-close this window normally to proceed to the following computations (by clicking the x at the top right)
-the second window to show up will be the gray histogram instersection resultant image
-close this window normally (by clicking the x at the top right)
-the third window to show up will be the gray chi squared measure resultant image
-close this window normally (by clicking the x at the top right)
-the fourth window to show up will be the color histogram instersection resultant image
-close this window normally (by clicking the x at the top right)
-the fifth window to show up will be the color chi squared measure resultant image
-close this window normally to end the program

/img/		//dir containing output of program (will populate after program completes)
/img/gray/	//dir of gray histograms (will populate after program completes)
/img/color/	//dir of color histograms (will populate after program completes)

/screenshots/	//dir of program results screenshots
